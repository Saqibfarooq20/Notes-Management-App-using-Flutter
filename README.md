# Notes Management App using Flutter

A simple Flutter app for creating and managing notes.

## Features
- Add new notes
- View notes in a list
- Edit existing notes
- Delete notes

## Tech Stack
- Flutter
- Dart

## Description
This project is a basic Notes Management App built using Flutter. It allows users to add, view, edit, and delete notes. The app was created to practice CRUD operations, screen navigation, ListView, and UI updates using setState.
